To use these files to create a new fresh project using Visual Studio 2022:
1. Delete all the old Visual Studio files and all the Bin folders.
2. Run the GenerateProjects.bat file.
3. Use Hazel.sln and open with VS2022.
4. Right click on the SandBox solution and choose this to start first.
5. Build Hazel.
6. Build Sandbox.
7. Clean and rebuild if necessary.
8. F5 Sandbox.
